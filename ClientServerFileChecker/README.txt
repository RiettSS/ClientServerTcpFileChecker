Маленький гайд по использованию программы:

1.Запустить приложение сервера и приложение клиента.
2.Запустить сервер на каком-то ip адресе (обычно локальный 127.0.0.1) с любым портом
3.Подключиться по этому адресу из приложения клиента.
4.Поиск файлов на сервере происходит по пути, например: вводим в поле path путь "С:\" и жмём кнопку Find.
После этого в окне под полем ввода появятся все папки и файлы, находящиеся по данному пути. Чтобы перейти
в какую-то папку нужно просто дописать её в поле path. Аналогично работает и весь остальной функционал программы:
загрузка файлов с сервера на клиент и обратно, проверка целостности по hash сумме.
